# 七、测试方案 & 优化评判标准（对接可视化平台）

## 7.1 对照测试组（控制变量法）
1. 基线组：FP16 + 无融合 + 传统KV + 单机
2. 实验组1：INT4量化 + 无融合 + 传统KV
3. 实验组2：量化 + 算子融合 + 传统KV
4. 实验组3：量化 + 融合 + 分页KV（完整单机优化）
5. 实验组4：全优化 + 分布式流水线

## 7.2 核心评判指标
1. 显存占用（量化、分页KV 评判）
2. 推理时延 / 生成速度 Token/s（算子融合、整体速度）
3. CPU 负载、网络延迟（分布式、通信评判）
4. 对话通顺度（量化精度评判）
5. 长时间运行稳定性（缓存、通信健壮性）

## 7.3 可视化评判形式
- 实时数字面板、动态折线图、多组对比柱状图
- 支持一键切换优化方案，即时观测指标变化

# 八、项目创新点 & 答辩配套文档（直接复用）

## 8.1 三大工程创新（写入报告/PPT）
1. **异构边缘终端自适应分层流水线架构**
   针对性能不均的校园笔记本做动态分层，搭配带心跳重连的TCP协议适配校园弱网。
2. **三重优化与分布式深度耦合**
   量化、算子融合、轻量化分页KV分别适配各节点，KV本地存储不跨传，降低网络压力。
3. **集成交互+监控+对照评测一体化可视化平台**
   自带控制变量实验能力，直观验证各优化效果。

整理之前所有核心问答：算子融合、QKV、量化、边缘设备、数据依赖、TCP粘包、KV不跨传原因等。
