## 概述

`test_db_config.py` 包含两类测试：
- **离线测试**（18个）：不需要数据库，验证配置解析、环境变量处理等
- **在线测试**（13个）：需要真实 PostgreSQL 数据库，验证 CRUD 操作、事务等

默认情况下，如果没有可用的 PostgreSQL 连接，在线测试会被跳过（显示为 `SKIPPED`）。

## 如何启用在线测试

### 方案 1：使用自动化脚本（推荐）

项目提供了自动化设置脚本：

```bash
# 1. 运行设置脚本（会自动创建测试数据库和用户）
python scripts/setup_test_db.py

# 2. 加载环境变量
# Windows PowerShell:
Get-Content .env.test | ForEach-Object { 
    $name, $value = $_.Split('='); 
    [Environment]::SetEnvironmentVariable($name, $value) 
}

# Linux/macOS:
export $(cat .env.test | xargs)

# 3. 运行测试
python -m pytest tests/test_db_config.py -v
```

### 方案 2：手动设置

#### 步骤 1：安装 PostgreSQL

**Windows (使用 Chocolatey):**
```bash
choco install postgresql14
net start postgresql-x64-14
```

**macOS (使用 Homebrew):**
```bash
brew install postgresql
brew services start postgresql
```

**Linux (Ubuntu/Debian):**
```bash
sudo apt update
sudo apt install postgresql postgresql-contrib
sudo systemctl start postgresql
sudo systemctl enable postgresql
```

#### 步骤 2：创建测试数据库

```bash
# 进入 PostgreSQL 命令行
psql -U postgres

# 在 psql 中执行以下命令：
CREATE DATABASE qlh_edge_inference_test;
CREATE USER qlh_test WITH PASSWORD 'test_password_123';
GRANT ALL PRIVILEGES ON DATABASE qlh_edge_inference_test TO qlh_test;
\q
```

#### 步骤 3：设置环境变量

**Windows PowerShell:**
```powershell
$env:QLH_DB_HOST="localhost"
$env:QLH_DB_PORT="5432"
$env:QLH_DB_NAME="qlh_edge_inference_test"
$env:QLH_DB_USER="qlh_test"
$env:QLH_DB_PASSWORD="test_password_123"
```

**Linux/macOS:**
```bash
export QLH_DB_HOST="localhost"
export QLH_DB_PORT="5432"
export QLH_DB_NAME="qlh_edge_inference_test"
export QLH_DB_USER="qlh_test"
export QLH_DB_PASSWORD="test_password_123"
```

#### 步骤 4：运行测试

```bash
python -m pytest tests/test_db_config.py -v
```

## 方案 3：使用 Docker（最简单）

如果不想在本机安装 PostgreSQL，可以使用 Docker：

```bash
# 启动 PostgreSQL 容器
docker run --name qlh-test-postgres \
  -e POSTGRES_DB=qlh_edge_inference_test \
  -e POSTGRES_USER=qlh_test \
  -e POSTGRES_PASSWORD=test_password_123 \
  -p 5432:5432 \
  -d postgres:14

# 等待容器启动（约 5 秒）
sleep 5

# 设置环境变量并运行测试
export QLH_DB_HOST="localhost"
export QLH_DB_PORT="5432"
export QLH_DB_NAME="qlh_edge_inference_test"
export QLH_DB_USER="qlh_test"
export QLH_DB_PASSWORD="test_password_123"

python -m pytest tests/test_db_config.py -v

# 测试完成后清理
docker stop qlh-test-postgres
docker rm qlh-test-postgres
```

## 环境变量说明

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `QLH_DB_HOST` | `localhost` | PostgreSQL 服务器地址 |
| `QLH_DB_PORT` | `5432` | PostgreSQL 端口（1-65535） |
| `QLH_DB_NAME` | `qlh_edge_inference` | 数据库名称 |
| `QLH_DB_USER` | `postgres` | 数据库用户名 |
| `QLH_DB_PASSWORD` | (空) | 数据库密码 |
| `QLH_DB_MIN_CONN` | `2` | 最小连接数（1-50） |
| `QLH_DB_MAX_CONN` | `8` | 最大连接数（1-100） |

## 常见问题

### Q1: 测试仍然被跳过，如何排查？

检查数据库连接：
```bash
# 手动测试连接
psql -h localhost -p 5432 -U qlh_test -d qlh_edge_inference_test
```

检查环境变量：
```bash
# Windows PowerShell
echo $env:QLH_DB_HOST
echo $env:QLH_DB_PORT
echo $env:QLH_DB_NAME

# Linux/macOS
echo $QLH_DB_HOST
echo $QLH_DB_PORT
echo $QLH_DB_NAME
```

### Q2: 权限错误

```bash
# 确保用户对数据库有完全权限
psql -U postgres -c "GRANT ALL PRIVILEGES ON DATABASE qlh_edge_inference_test TO qlh_test;"
psql -U postgres -d qlh_edge_inference_test -c "GRANT ALL ON SCHEMA public TO qlh_test;"
```

### Q3: 连接超时

```bash
# 检查 PostgreSQL 是否运行
# Windows:
Get-Service postgresql*

# Linux:
sudo systemctl status postgresql

# macOS:
brew services list
```

## CI/CD 集成

在 GitHub Actions 中使用 PostgreSQL 服务：

```yaml
name: Database Tests

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    
    services:
      postgres:
        image: postgres:14
        env:
          POSTGRES_DB: qlh_edge_inference_test
          POSTGRES_USER: qlh_test
          POSTGRES_PASSWORD: test_password_123
        ports:
          - 5432:5432
        options: >-
          --health-cmd pg_isready
          --health-interval 10s
          --health-timeout 5s
          --health-retries 5
    
    steps:
      - uses: actions/checkout@v3
      
      - name: Set up Python
        uses: actions/setup-python@v4
        with:
          python-version: '3.12'
      
      - name: Install dependencies
        run: pip install -r requirements.txt
      
      - name: Run database tests
        env:
          QLH_DB_HOST: localhost
          QLH_DB_PORT: 5432
          QLH_DB_NAME: qlh_edge_inference_test
          QLH_DB_USER: qlh_test
          QLH_DB_PASSWORD: test_password_123
        run: python -m pytest tests/test_db_config.py -v
```

## 总结

- **无数据库环境**：18个离线测试全部通过，13个在线测试被跳过
- **有数据库环境**：31个测试全部通过（包括13个在线测试）
- **推荐方案**：使用 `scripts/setup_test_db.py` 自动化脚本或 Docker
- **CI/CD**：使用 GitHub Actions 的 PostgreSQL 服务容器
